# Amélioration du suivi d'activité des entrepôts

##  Contexte

Dans le cadre d'une mission de Data Analyst Supply Chain, ce projet consiste à analyser les activités d'un entrepôt afin d'améliorer le suivi des opérations logistiques grâce à la création d'indicateurs de performance et d'un tableau de bord décisionnel.

L'objectif est de transformer des données brutes en informations claires permettant aux équipes de piloter efficacement les stocks et les activités de l'entrepôt.

---

## Objectifs

- Nettoyer et préparer les données avec Power Query.
- Structurer les données pour faciliter leur exploitation.
- Construire des Tableaux Croisés Dynamiques (TCD).
- Créer des indicateurs clés de performance (KPI).
- Concevoir un tableau de bord interactif.
- Identifier les tendances et les anomalies.
- Proposer des recommandations pour améliorer le pilotage logistique.

---

## Outils utilisés

- Microsoft Excel
- Power Query
- Tableaux Croisés Dynamiques (TCD)
- Graphiques Excel
- Tableaux de bord

---

## Méthodologie

### 1. Préparation des données

- Importation des données.
- Nettoyage des informations avec Power Query.
- Vérification des incohérences.
- Structuration des tables.
- Fusionner des tables

### 2. Analyse des données

- Création de Tableaux Croisés Dynamiques.
- Calcul des indicateurs de performance.
- Analyse des mouvements et de l'activité des entrepôts.

### 3. Visualisation

Création d'un tableau de bord regroupant les principaux indicateurs afin de faciliter la prise de décision.

---

## Résultats

Cette analyse permet notamment de :

- suivre l'activité des entrepôts en temps réel ;
- visualiser les principaux KPI logistiques ;
- identifier rapidement les écarts de performance ;
- améliorer le pilotage opérationnel grâce à un tableau de bord synthétique.

---

##  Recommandations

- Automatiser l'actualisation des données avec Power Query.
- Suivre régulièrement les indicateurs de performance.
- Mettre en place des seuils d'alerte sur les KPI critiques.
- Utiliser le tableau de bord comme outil d'aide à la décision.
